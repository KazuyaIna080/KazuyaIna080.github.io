## 概要

本プロジェクトは、認知乖離分析（Cognitive Divergence Analysis: CDA）フレームワークを提示する。

CDAは「なぜ、平和的解決という選択肢が、意思決定の場から消えていくのか」という問いに答えるため、以下の枠組みを提供する：

### PoNR三条件

| 条件 | 定義 | 認知状態 |
|------|------|---------|
| **Choice Collapse** | 平和的選択肢の認知上の消滅 | 「他に道がない」 |
| **Time Inversion** | 時間認知の反転 | 「待てば不利になる」 |
| **Legitimacy via Harm** | 加害行為が正統性を生成 | 「やらねば正統性を失う」 |

### 戦争の三類型

| 類型 | 特徴 | 事例 |
|------|------|------|
| **漂流相（Drift Phase）** | 選択肢が段階的に消滅 | 日本1941、WWI 1914 |
| **決定相（Decision Phase）** | 三条件が瞬時に同時成立 | アメリカ1941（真珠湾後） |
| **巻き添え相（Vent Phase）** | 置換標的として選定 | ウクライナ、ガザ |

## 核心的主張

> **問題は、アクターが戦争を選ぶことではない。構造が選択肢を消すことである。**
>
> 構造が見えれば、構造は変えられる。構造が見えなければ、構造が我々を変える。
